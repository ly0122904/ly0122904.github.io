Books Search Library

Using the Open Library Search API our team make an application that allow the user to search for books in the Open Library database. Users will be able to select a book and find more information such as the title, the author, the first publish year, and an image of the cover. The application use a list view on the left side of the page which after the search button is clicked will fill with titles of different books that match the search terms within the Open Library database. There will also be an about page which will list all the team members names as well as any other important information.

2D Predator-Prey Simulation with C++

The goal for this programming project is to create a simple modified 2D predator-prey simulation (aka a zombie apocalypse). In this simulation the prey are humans and the predators are zombies. These beings live in a 20 x 20 grid of cells (a city). Only one being may occupy a cell at a time. The grid is enclosed, so a being is not allowed to move off the edges of the city or appear on the other side. Time is simulated in steps.
